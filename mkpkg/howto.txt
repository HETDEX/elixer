1. from directory above elixer 
  - add __init__.py
  - add setup.py
  - add MANIFEST.in

(setup has version info, etc)


2. run > python setup.py sdist bdist_wheel
  - creates build, dist, hedex_elixer.egg-info

3. copy dist/*.gz  (that is what you will install)

4. run > pip install --user hetdex-elixer-1.6.0.tar.gz
